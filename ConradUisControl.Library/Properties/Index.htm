Welcome to the CUC frontend

You'll find information on the usage of the CUC here.

Configuration

Find the ConradUisControl.exe.config file in your bin-directory. Open it with any text editor. You then see the contents in an XML-file, where the <appSettings>-section represents the configuration.
Descriptions:

  • listenPort: the port on the local machine where the webserver shall wait for GET requests
  • deviceIp: the IP of the device within your network
  • devicePort: the port of the device
  • username: used for authentication on the device
  • password: used for authentication on the device
  • outletCount: how many outlets the device features (starting at 1)

API

The CUC web server allows access to some methods, which can be used to set the outlet status.

Name Type Parameters Example
enable GET/POST outlet index /enable?1
disable GET/POST outlet index /disable?1
