Key   | Function:
------+--------------------------
C     | Clear screen
H     | Display this text
Num0  | Read outlet status
ESC   | Quit
------+--------------------------
Num1  | Set outlet #1 to ENABLED
Num2  | Set outlet #1 to DISABLED
------+--------------------------
Num4  | Set outlet #2 to ENABLED
Num5  | Set outlet #2 to DISABLED
